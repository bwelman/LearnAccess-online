Task: Generated report

An automatically generated report with a manual adjustment.

Access can automatically generate a report on the basis of a table or a query. Sometimes the generated report is sufficient, but mostly you nee to made manual adjustments.
  1. Open database candy2016.accdb.
  2. Select query Sales per region per box.
  3. Click tab Create > Report (group Reports).
    The report is generated and opened in Layout View. Not beautiful is that the value of Region is repeated for each record and that the Sales values are not properly formatted. In the following steps this will be changed.
  4. Close the report and answer the question to save the changes with Yes.
    The dialog box Save As is displayed.
  5. Name the report Sales per region per box and click OK.
  6. Open the report Sales per region per box in Design View.
  7. Select in section Detail the field Region. In the Property Sheet change the value of property Hide Duplicates in Yes.
    Tip: The Property Sheet is on the right side of the screen and can be made visible and invisible by clicking on button Property Sheet or by using hotkey F4.
  8. Select in section Detail the field Sales. In the Property Sheet set the value of property Format on Currency.
  9. Switch to Report View.
    The value of the Region field is now displayed only once and the sales values are in currency format.
  10. Close the report and save the changes.
