Task: Creating Labels

How to create labels with the Wizard.

In this task you are going to create address labels for the customers with the Label Wizard. First you have to select the table with the necessary fields.
  1. Open database candy2016.accdb.
  2. Select table Customers.
  3. Choose tab Create > Labels (group Reports.
  4. Select measures Metric, manufacturer Avery and then product Avery L7160.
  5. Click Next.

    In the screen that is displayed now you can change font and color for the text.

  6. Accept the default settings and click Next.
    By double clicking on a field name, the field is inserted at the cursor location. You can also use button>. The field name appears between brackets on the prototype label. Also, text and spaces can be typed. With the Enter button you can create a new line.
  7. Create the following prototype label: 
    {FirstName} {LastName}
{Address}
{ZipCode}  {City}

    Between first and last name is one space, and between zip code and city are two spaces.

  8. Click op Next.

    In the screen that is displayed now you can specify if the labels should be sorted, and if so, by which fields.

  9. There must be sorted by ZipCode. Add this field and click Next.

    The last screen of the Wizard is displayed now. Here you can specify the name for the report.

  10. Name the report Labels Customers and click Finish.
  11. Close the report.
