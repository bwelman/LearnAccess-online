Task: Parameter query

A query that asks you for entering a value during execution.

A parameter query is a query that displays a dialog box that prompts the user to provide additional information during the execution, such as criteria for retrieving records or a value that you want to insert into a field. You can design the query so that multiple data is requested, for example a start and end date. Then all the records can be retrieved with dates between them.

Parameter queries are also easy as a basis for forms and reports. Based on a parameter query, you can for example create a monthly revenue report. When printing the report you are prompted by a dialog box which month you want to print, you enter the month and then the correct report will be printed.

INFORMATION NEEDS

In the company Snoopy one gets regular customer questions over the phone about a particular order. You want to quickly be able to answer such a question. The goal now is to get the data from a particular order quickly on the screen. That is possible with a parameter query, asking for the order code during the execution of the query.

ANALYSIS

The necessary information about a particular order are in the tables Orders and OrderDetails. Questions about the order code can be controlled via a criterion.

  1. Open database candy2016.accdb.
  2. Choose tab Create > Query Design (group Queries).
    Access creates a new blank query window and displays the dialog box Show Table.
  3. Add tables Orders and OrderDetails and then click Close.
    The tables have been added to the query window.
  4. Add successively the fields OrderCode, CustomerCode, OrderDate (from Orders) and BoxCode en Quantity (from OrderDetails) to grid by double clicking on the field.
  5. Click in the box Criteria under column OrderCode and type [Enter order code].
  6. Switch to Datasheet View.
    The dialog box Enter Parameter Value appears and ask you to enter the order code..
  7. Enter a value, e.g. 30 and click OK.
  8. Close the query and answer the question to save the changes with Yes.
  9. Name the query Information specific order and click OK.
