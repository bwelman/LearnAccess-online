Task: Make Table query

Creating a new table with the Make Table query.

INFORMATION NEEDS

The marketing department has a special offer for all customers in Friesland. For this, they need a table with only the customer data for customers in the province Friesland.

ANALYSIS

All necessary data is in table Customers. Selecting for Friesland can be done with a criterion in the query. First, you need to create this select query and then change the query type to Make Table.

  1. Open database candy2016.accdb.
Creating Select query
  1. Choose tab Create > Query Design (group Queries).
    Access creates a new blank query window and displays the dialog box Show Table.
  2. Add table Customers and then click Close.
    The table has been added to the query window.
  3. Add all fields by double-clicking on the asterisk (*) and then add separate again the field Province.
  4. Enter as criterion "FR" for the field Province and don't show this extra field, because it is in fact already shown by Customers.
  5. Check the output of the query with the Datasheet View.
  6. Save the query with the name Customers Friesland.
Change the query type
  1. Open the query in Design View and click Make Table (group Query Type).
    The dialog box Make Table is displayed.
  2. Name the new table Friesland customers and select that this table should come in the current database.
  3. Click OK.
  4. Click tab Design > Run (group Results).
    A dialog appears asking for confirmation.
  5. Click Yes.
    The new table is created.
  6. Close query
    A dialog box asking whether the changes should be saved.
    Note: This is because the query type has been changed. This is also seen in the navigation pane under Queries. The query icon indicates an action query:
  7. Click Yes.
