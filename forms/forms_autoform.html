Task: Generated form

Create a generated form with manual adjustments.

When new records must be added to a table, it is usually intended that all of the fields get a value. Such a form must contain all the fields from the table. The quickest way to create such a form is by first selecting the table and then through the command Form automatically generate a form containing all fields. After that, you can manually adjust the form.

In this task you create a form for a warehouse employee to modify the warehouse stock of a box and eventual to indicate if a box is out of production. It shouldn’t be possible for the emplyee to change the other box data with this form. The form should look something like this:

  1. Open database candy2016.accdb.
  2. Select table Boxes. It's not necessary to open the table.
  3. Click tab Create > Form (group Forms).

    The form is created and displayed in Layout View.

If Access finds a table that has a one-to-many relationship with the table or query that you used to create the form, Access adds a subdatasheet to the form that is based on the related table or query. In this case the form is based on the table Boxes, and there is a one-to-many relationship defined between the table Boxes and the table OrderDetails. The subdatasheet displays all the records in the OrderDetails table that pertain to the current Boxes record. If you decide that you do not want the subdatasheet on the form, you can delete the subdatasheet in the Layout view by selecting the datasheet, and then pressing Delete. If there is more than one table with a one-to-many relationship to the table that you used to create the form, Access does not add any datasheets to the form.

  1. Select the subdatasheet and press Delete.
  2. Switch to Design View.

In a form which has been created through this method, two controls are created for each field: a Text Box and a linked Label.

All controls are arranged vertically with a label to the left of the control. This is called a Stacked layout. That's why you can't move a control anywhere on the form. If this is the intention, then you must remove the layout for that control through tab Arrange > Remove Layout (group Table).

  1. Select the text in frame Boxes in the Form Header and change it in Stock-in-trade. Change the properties Font Weight in Bold and Font Size in 20.
    Tip: The Property Sheet is on the right side of the screen and can be made visible and invisible by clicking on button Property Sheet or by using hotkey F4. And to faster find a specific property, you can alphabetically sort the list with the button in the top right corner of the sheet.
  2. Select the text boxes of all fields and change the property Width in 3". After that deselect by clicking elsewhere in the form.

The Locked property of a control can be used to protect data in a field by making it read-only.

  1. Select all text boxes except Stock and OutOfProduction and change property Locked (tab Data) in Yes.
  2. With the text boxes selected change the background color from white in light grey. You can do this in one of the following ways:
    • Change property Back Color.
    • Right click on one of the selected fields and choose from the shortcut menu Fill/Back Color.
  3. Switch to Form View to check the result.
  4. Close the form the form, answer the question to save the changes with Yes.
  5. Name the form Stock-in-trade and click OK.
