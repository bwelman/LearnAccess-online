Exercise: form003 - Boxes per customer

Create a main form with subform in which the main form displays the box data and the subform displays the total sales per customer for this box. For this, first create a query that calculates the total number of ordered boxes per customer. Name the query Boxes per customer and the forms form003 main and form003 sub.
