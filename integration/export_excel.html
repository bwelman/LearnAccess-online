Task: Export to Excel

How to export an Access table to a new Excel document.

Je kunt eenvoudig een tabel vanuit Access naar Excel exporteren.
  1. Open database candy2016.accdb.
  2. Select table Boxes.
  3. Right click on the table name and choose Export > Excel.
    The dialog box Export - Excel Spreadsheet is displayed.
  4. Specify file name and path, select option Export data with formatting and layout and click OK.
    The dialog box Save Export Steps is displayed.
  5. It's not necessary to save the export steps. Click Close.
  6. Check the export by opening the file in Excel.
