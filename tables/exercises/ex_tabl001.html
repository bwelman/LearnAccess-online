Exercise: tabl001 - Prevent duplicate names

The table Suppliers has a field Supplier. This is the name of the supplier. It is now possible to use the same supplier name more then once, which is undesirable. To prevent this, you can set the value of the property Indexed of this field on Yes (No Duplicates). Apply this change.
